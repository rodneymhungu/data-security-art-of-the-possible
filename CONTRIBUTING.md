# Contributing

Thanks for taking the time to help keep this reference accurate and useful.

This repository hosts a public version of Microsoft's "Art of the Possible" data security deck. The content is Microsoft's work; this version is curated and maintained for partner self service.

## How to contribute

Contributions are handled through issues.

- **Out of date content.** Microsoft Purview moves quickly. If a capability shown has changed, moved, or been renamed, open an issue describing what you saw and what it is now. A link to the relevant Microsoft Learn page is helpful.
- **Suggest a scenario.** If there is a customer scenario worth adding, open an issue outlining the problem, the Purview capabilities involved, and why it would help readers.
- **Corrections.** Typos, broken navigation links, or factual errors are all worth raising.

Please use the issue template where one fits. It helps gather the detail needed to act on a report.

## Scope

This is positioning and reference material, not deployment guidance. Suggestions that keep it accurate, current, and easy to navigate are the most useful.

## Provenance

The deck originates from Microsoft. No ownership or licence is asserted over the Microsoft content. Please keep that in mind when proposing changes, and do not submit material that would misrepresent the source.
